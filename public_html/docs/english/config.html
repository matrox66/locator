glFusion Locator Plugin Configuration

The Locator Plugin allows your site to provide a locator function, such as a store locator or points of interest. Visitors can search for locations within a selected radius of some predefined locations, or within a given radius from their home location.

This version requires GLFusion 1.4.0 or higher.

Main Locator Options

Default Radius
The default search radius.
Default Distance Unit
The default distance unit used in searches. Users may override this for each search.
Show Map?
Selects whether a Google map will be shown in the location information view.
Use submission queue
Set this to "Yes" if user submissions should be moderated. If this is "No", user-submitted locations will be publically available as soon as they are submitted.
Allow Anonymous or Member Submissions?
Select whether submissions are accepted from site members, or anonymous users and members, or only admins.
Show Right Blocks?
Set this to "Yes" to display the right blocks when this plugin is active. Setting this to "No" prevents the right blocks from being displayed, giving more space to the plugin.
Days after which user-entered locations are purged
User-entered search origins are cached in a database table. This value indicates the number of days after which these free-form user entries are removed.

Note that the database entries associated with the "Location" field in the user account settings are not purged. This refers only to the free-form entries created on the Locator home page.

Show maps in user profiles?
Set this to "Yes" to have maps displayed with user profiles, using the user-supplied location value.
Show menu option on user menu?
Set this to "Yes" to have a menu option for the plugin displayed on the built-in "plugins" menu.
Integrate with the Weather plugin?
If this value is "Yes", and the Weather plugin is installed and enabled, then local weather for locations will be displayed when the location is viewed. If the Weather plugin is not installed or is not enabled, this setting has no effect.
Add a form to get directions?
If this value is "Yes", then a form will be included on the Location Detail page to allow the user to enter a starting address and get directions from Google Maps. If this is "No", then the directions form is suppressed.
Operate in API-Only mode?
If this value is "Yes", then the plugin will not be available as a guest-facing page but only as a provider to other plugins, e.g. Evlist. Set this to "No" if you wish to make the plugin available to your site visitors.

Geocoding Section

Auto-Fill Location Coordinates?
If this is "Yes", then the plugin will attempt to determine the location for newly-submitted locations using the Google Map API. Otherwise, the coordinates must be entered manually.
Google API Key
Enter your own Google Map API key here. This is required in order to use Google to automatically determine the geographic coordinates for locations.

visit http://code.google.com/apis/maps/ to get started.

URL for Geocoding
This is the URL to the geocoding function of Google Maps. The plugin will substitute the actuall data in place of these placeholders:
%address% - The address of the location
%google_key% - Your Google Maps API key

Permissions

Default Group
This is the default group which will be assigned to the "group_id" value for newly-submitted locations. The "Default Permission[1]" selected will be applied as well to give read and edit permission to the location.
